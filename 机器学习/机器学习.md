### 创建虚拟环境

```
# win10创建虚拟环境
virtualenv -p d:\softwareintsall\python\python.exe virtualenv1
# 启动虚拟环境时需要先切换到虚拟环境的Scripts目录下再执行如下命令。
activate
#退出虚拟环境
deactivate 
```

#### tf-idf含义

```python
# tf: term frequency:词的频率
# idf: inverse document frequency:逆文档频率 == log(总文档数量/该词出现的文档数量)
# tf*idf:重要性程度
```
